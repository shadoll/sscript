# sscript
Useful Shell scripts for projects

# Install

with npm

`npm install sscripts`

with yarn

`yarn add sscripts`

# Run

shell

`./node_modules/.bin/{script_name}`

npm

`npm run {script_name}`

yarn 

`yarn {script_name}`
